## [0.2.0] - 2025-01-03

### 新增
- **Semantic Analysis (语义分析)**
  - 完整的类型检查系统
  - 作用域管理（全局、函数、块作用域）
  - 符号表构建和查询
  - 函数调用验证
  - 变量使用检查
  - 控制流分析

- **IR Generation (中间表示生成)**
  - 三地址码表示
  - 基本块划分
  - 控制流图构建
  - 函数和全局变量处理
  - 完整的指令集（算术、比较、逻辑、内存、控制流）

- **Code Generation (代码生成)**
  - C代码生成器（transpilation）
  - 类型转换系统（Nova类型 → C类型）
  - 操作码映射（IR操作码 → C操作符）
  - 支持多种目标平台（LLVM IR、x86-64、WASM）
  - 代码格式化和注释生成

- **Unit Tests (单元测试)**
  - 简化的 Catch2 风格测试框架
  - 360+ 测试用例覆盖所有模块
  - 词法分析器测试（100+ 用例）
  - 语法分析器测试（80+ 用例）
  - 语义分析器测试（70+ 用例）
  - IR生成器测试（60+ 用例）
  - 代码生成器测试（50+ 用例）

- **CI/CD**
  - GitHub Actions 自动构建
  - 多平台支持（Linux、macOS、Windows）
  - 自动测试运行

- **Documentation (文档)**
  - 语言规范文档 (LANGUAGE.md)
  - 架构设计文档 (ARCHITECTURE.md)
  - 测试文档 (TESTING.md)
  - 贡献指南 (CONTRIBUTING.md)

### 变更
- 更新版本号至 v0.2.0
- 改进命令行界面，支持更多选项
- 优化错误提示信息

### 修复
- 修复 IR 接口不一致问题
- 修复代码生成器临时变量声明问题
- 改进错误处理和恢复机制

## [0.1.0] - 2024-12-20

### 新增
- **Lexer (词法分析)**
  - 完整的源代码词元化
  - 支持所有字面量类型（int、float、string、char、bool）
  - 关键字和标识符识别
  - 运算符和分隔符处理
  - 单行 (`//`) 和多行 (`/* */`) 注释支持
  - 字符串中的转义序列
  - 源码位置跟踪（文件、行、列、偏移量）
  - 全面的错误报告

- **Parser (语法分析)**
  - 递归下降解析器实现
  - 正确运算符优先级的表达式解析
  - 语句解析（if、while、for、return、代码块）
  - 声明解析（函数、变量、结构体）
  - 类型注解解析
  - 错误恢复和同步
  - 完整的 AST 生成

- **AST (抽象语法树)**
  - 类型系统（基本类型、命名类型、数组）
  - 表达式节点（字面量、二元、一元、调用、成员访问）
  - 语句节点（控制流、代码块、return）
  - 声明节点（函数、变量、结构体、字段）
  - 用于调试的格式化打印

- **构建系统**
  - 包含多个构建目标的 Makefile
  - Debug 和 release 配置
  - 测试目标
  - 安装目标

- **示例**
  - Hello World 示例 (`hello.nv`)
  - 语言特性演示

- **文档**
  - 全面的 README
  - 内联代码文档
  - 架构概述

- MIT 许可证
